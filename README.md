#  Unity Networking System

##  프로젝트 개요
- **장르**: 멀티플레이 타임어택
- **개발 툴**: C#, Unity
- **플랫폼**: PC (Windows)
- **개발 기간**: 2025.01.06 ~ 2025.02.05
- **팀 구성**: 5인 (기획, 개발, 디자인 등)

**Tech: Unity, C#, Photon PUN**

## 개발 목적
- Photon PUN 기반 멀티플레이 네트워크 시스템 구현
- Player 생성 및 Network Object 동기화 구조 설계
- Room 기반 매칭 시스템 및 게임 입장 로직 구현
- 캐주얼 게임 구조를 활용한 멀티플레이 게임 흐름 구현

## 게임 컨셉
- 장애물을 회피하며 결승점에 도달하는 캐주얼 경쟁 게임
- 여러 플레이어가 동시에 참여하는 멀티플레이 구조
- Photon Network 기반 Player 상태 동기화
- 간단한 조작을 통한 접근성 높은 캐주얼 게임 플레이

##  역할
- **팀장 및 PM 역할 수행: 전체 기획 수립, 일정 조율, 테스트 및 피드백**
- **맵 구조 기획 및 레벨 디자인**
- **상호작용 요소 개발**
- **캐릭터 리스폰 및 게임 흐름 제어**
- **UI 및 사운드 매니저 구현**
